How Not to Be a Native Bayesian

It's widely claimed that intelligence comes largely from working memory. This is not a trivial suggestion. Working memory is a central part of human and animal intellect. In the 1980s there was general interest in the working memory concept, but it hasn't been adequately measured. Many early researchers had no interest in measuring things and had nothing better to measure.


However, some interesting research was done in the 1990s by Stanislas Dehaene, who discovered that you can control your working memory by changing the kind of things you're trying to remember. If you're trying to recall an abstract object with no associated sensory stimulus, your working memory holds a few objects. If you're trying something visual and associative, it holds a single item. By consciously applying this heuristic to how many items you'll keep in mind you can radically improve your memory storage capacities.


A recent paper by Dehaene explores whether this process generalizes. To test this theory, he had subjects remember pairs of names and faces, but then had subjects guess whether faces were related to the names or vice versa using a picture/\xe2\x80\x8bstory test. Half the subjects were asked to name the face when they were looking at the face and vice versa. The other half were told to name the face whenever they could guess the name from the face. The second group did better at the image/\xe2\x80\x8bstory test, but not much better. Dehaene concluded that the second group used a strategy involving working memory of visual information.


It's an interesting observation that our brains store different kinds of information in our working memory in different ways. The more you consciously work with your working memory, the better you can improve it. Therefore, when you work with your working memories and consciously change what kind of working memory you're using, you can dramatically improve your short-term working memory of visual and associative information. This process can improve your performance on certain kinds of cognitive tests while you're not trying, although not as much. Dehaene calls this "the Leech Book of Working Memory".

The Three Kinds of Information


Babble and prune can be divided up into three types of information.


A short list of the three kinds can go like this:


  • Visual information

  • Associative information with some associated sensory stimuli

  • Working memory (aka memory palace, mnemonic, symbol memory, short-term memory, working memory, etc.)

A lot of visual and associational information will be in working memory. A lot of other information isn't in working memory, but will become so if you're allowed time to think about it.


Visual Babble: Your working memory holds a short clip of the kind of thing you're visualizing.


Associative Babble: Your working memory holds the abstract idea plus a link to a sensory stimulus or a sound, the meaning of which is in your short-term memory.


Working Memory Babble: Working memory holds a short string of letters and the link to another string of letters.


Prune: This is done in a special way by linking them to items in your working memory. This allows the information to be stored without requiring you to wait until something happens in your working memory to bring it to your attention.
